# Unisystem – Konsollbasert universitetssystem

Dette prosjektet er et enkelt, konsollbasert universitetssystem skrevet i C#.  
Systemet lar deg registrere og administrere studenter, inkludert utvekslingsstudenter, fag og annen relevant informasjon.

##  Funksjoner
- Registrere nye studenter
- Registrere utvekslingsstudenter med hjemuniversitet, land og periode
- Vise oversikt over registrerte studenter
- Menybasert navigasjon i konsollen
- Enkel og oversiktlig struktur med egne klasser for modeller
