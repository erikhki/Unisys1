# Unisystem – Konsollbasert universitetssystem

Dette prosjektet er et enkelt, konsollbasert universitetssystem skrevet i C#.  
Systemet lar deg registrere og administrere studenter, inkludert utvekslingsstudenter, fag og annen relevant informasjon.

## ✨ Funksjoner
- Registrere nye studenter
- Registrere utvekslingsstudenter med hjemuniversitet, land og periode
- Vise oversikt over registrerte studenter
- Menybasert navigasjon i konsollen
- Enkel og oversiktlig struktur med egne klasser for modeller

## 🧱 Struktur
Prosjektet er organisert slik:
- `Program.cs` – hovedmeny og programflyt
- `Modeller/` – inneholder klasser som `Student`, `Utvekslingsstudenter`, osv.
- `Services/` (hvis du har det) – logikk for håndtering av data

## 🛠️ Teknologi
- Språk: C#
- Plattform: .NET (konsollapplikasjon)
- IDE: Visual Studio

## 📚 Formål
Prosjektet er laget som en del av et skolearbeid for å øve på:
- Objektorientert programmering
- Arv og modellering av datastrukturer
- Meny- og kontrollflyt i konsollapplikasjoner
- Strukturering av kode i mapper og klasser
